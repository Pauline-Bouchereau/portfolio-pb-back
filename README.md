### 🚀 Running the project

#### Clone this repository :

```javascript
git clone https://github.com/Pauline-Bouchereau/portfolio-pb-back
cd portfolio-pb-back
```

#### Install packages :

```javascript
npm install
```

#### When installation is complete :

```javascript
npx nodemon
```
